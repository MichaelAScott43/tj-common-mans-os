# STEADY MVP ROADMAP

## PHASE 1 - FOUNDATION
Goal: establish identity and emotional experience.

### Deliverables
- Rebrand app as Steady
- TJ as practical assistant
- Arlane support mode
- Mobile responsive UI
- Landing page refresh
- Character switcher
- Waitlist capture

## PHASE 2 - CORE FUNCTIONALITY
Goal: make Steady genuinely useful every day.

### TJ Features
- reminders
- errands
- affordability comparisons
- message drafting
- appointment prep
- organization help
- task planning

### Arlane Features
- grounding conversations
- encouragement
- stress decompression
- calm support flows
- emotional check-ins

## PHASE 3 - LIFE NAVIGATION
Goal: become everyday support infrastructure.

### Features
- life admin help
- budgeting assistance
- resource discovery
- local recommendations
- affordability engine
- workflow simplification

## SAFETY PRINCIPLES
Steady is not:
- therapy
- medical treatment
- crisis intervention
- legal advice

Steady always:
- asks for confirmation before actions
- protects user privacy
- avoids manipulative behavior
- stays emotionally grounded

## LONG-TERM VISION
Become the trusted AI sidekick for the working man and woman.
