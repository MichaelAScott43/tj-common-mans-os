# Steady App

## Goal
A mobile-first mental health support app for iOS and Android that is useful, calm, and not flashy.

## Users
- Veterans
- Civilians dealing with stress, anxiety, burnout

## Core Features
- Daily check-in (1–5 mood scale)
- Journal with prompts
- AI reflection summaries
- "Help Now" button
- Safety Plan (trusted contacts)
- Pattern tracking

## Help Button
3 options:
1. Calm Me Now
2. Contact Someone I Trust
3. Crisis Support Now

## Tracking
- No always-on tracking
- Optional location sharing ONLY when user chooses it
- Only sent to trusted contact

## Global
- Country selection
- Resource packs per country
- Veteran mode toggle

## Veteran Mode
- Show veteran-specific resources
- Show US flag 🇺🇸 ONLY for US veteran resources
