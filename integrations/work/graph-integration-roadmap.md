1. Register Azure AD app with delegated scopes for mail/calendar metadata.
2. Implement OAuth authorization code flow with PKCE.

5. Add optional user opt-in for body/content retrieval.
6. Introduce Work Mode policy controls and admin consent checks.
7. Package Teams app + Outlook add-in with stable production IDs.
