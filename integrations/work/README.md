# Work Integrations Placeholder Architecture

This directory provides the scaffold for future Work Mode integrations.

## Included placeholders
- `teams.manifest.json`: starter Microsoft Teams app manifest.
- `outlook-addin.manifest.xml`: starter Outlook/Office add-in manifest.
- `graph-integration-roadmap.md`: implementation plan for Microsoft Graph + Teams Toolkit.

## Privacy boundaries
- Work account data is isolated from Personal Mode data.
- All work integrations are optional and token-based.
- Metadata-only synchronization should be the default.
